What is an IFSC Code?

IFSC stands for Indian Financial System Code. It is an 11-character alphanumeric code assigned by the Reserve Bank of India (RBI) to uniquely identify every bank branch participating in the electronic fund transfer network in India.

IFSC codes are mandatory for all digital money transfers including NEFT (National Electronic Funds Transfer), RTGS (Real Time Gross Settlement), and IMPS (Immediate Payment Service).

IFSC Code Format — Decoded

SBIN
Bank Code (4 chars)
0
Reserved
001234
Branch Code (6 chars)

Where to Find Your IFSC Code

  • Cheque book: Printed at the top of every cheque leaf
  • Bank passbook: Printed on the first page with account details
  • Internet banking: Under account details or profile section
  • Bank's mobile app: In account information section
  • Bank statement: Listed alongside your account number
  • RBI website: Official list at rbi.org.in

Frequently Asked Questions

What is an IFSC code and what is it used for? +
IFSC (Indian Financial System Code) is an 11-character code that uniquely identifies every bank branch in India. It is mandatory for NEFT, RTGS, and IMPS electronic fund transfers. Without the correct IFSC, you cannot complete an online bank transfer.
Is IFSC code the same for all branches of a bank? +
No. Each branch has its own unique IFSC code. For example, SBI Andheri East (Mumbai) and SBI Connaught Place (Delhi) have completely different IFSC codes even though both belong to State Bank of India. Always use the branch-specific IFSC code.
What is the IFSC code of SBI main branch? +
SBI has thousands of branches across India, each with its own IFSC. The SBI Mumbai Main Branch IFSC is SBIN0000300. For any other SBI branch, use our finder above — select State Bank of India, choose your state and branch to get the exact code.
What is the difference between IFSC and MICR code? +
IFSC is an 11-character alphanumeric code used for online electronic transfers (NEFT, RTGS, IMPS). MICR (Magnetic Ink Character Recognition) is a 9-digit numeric code printed on cheques using magnetic ink, used specifically for cheque processing. Both identify bank branches but for different transaction types.
Can an IFSC code change? +
Yes, IFSC codes can change when banks merge (e.g., after the merger of Vijaya Bank and Dena Bank into Bank of Baroda, old IFSC codes were phased out). When a branch closes or merges with another, its IFSC may be reassigned. Always verify the IFSC for important transfers.
How do I find IFSC code from account number? +
You cannot determine the IFSC code from an account number alone. The IFSC is branch-specific and not encoded in the account number. You can find it on your cheque book, passbook, internet banking portal, or by using our IFSC Finder above by searching for your bank and branch.
